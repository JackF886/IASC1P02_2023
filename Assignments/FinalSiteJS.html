JavaScript Introduction

Syntax

While Java and JavaScript are very different languages, it is still possible to draw similarities between their syntax's. The main difference is the typing of variables. JavaScript heavily relies on logical operators and the use of variables to both store input and perform operations/calculations. While simple in theory, the structure allows for looping and methods which aim to reduce the amount of code used and reducing complexity too. Like CSS, semicolons are used to end lines of code and round brackets are for passing values into functions and comparators.

Methods and Functions

  • getElementByID: Used as a selector to target an HTML element by their ID
  • function methodName(){: The method declaration used to fire JS when the function is called; Typically through a button
  • .innerHTML: Targets the content within the HTML tags and can change content in real-time
  • Important Concepts
    • getMethods: Returns the value of some variable to be used in calculations or another function
    • Variable Assignments: Giving the variable a value and typing, whether explicitly or implicitly casted
  • Variables: A named cell that stores a value. Often used to compare with other variables and perform calculations
  • Method Parameters: Variables and values passed into a method or function so it can be used locally

Variables

Variables are an essential part of JavaScript and coding in general. They are empty storage cells that when defined, can hold some value. These values are can be manually assigned or updated by the programmer and given a type. While variables can have different types, these types hold different properties and functions too. Most commonly, String and Number variables are used to interact with one another. As a result, problems such as calculations can be solved using these variables and increase the generality while increasing the functionality of code too.

Progress Bar Code
function animateProgressBar() {
    var progressBar =
    document.getElementById("progress-bar");
    var duration = 4000; // 4 seconds
    var interval = 50;

    var start = null;
    function step(timestamp) {
        if (!start) start = timestamp;

        var progress = timestamp - start;
        progressBar.style.width =
          (progress / duration) * 100 + "%";

        if (progress < duration) {
            requestAnimationFrame(step);
        }
    }
    requestAnimationFrame(step);
}
